HOME PAGE
A website is a collection of files accessed through a web address, covering a particular theme or subject, and managed by a particular person or organization. Its opening page is called a homepage.

A website resides on servers connected to the web network and is able to format and send information requested by worldwide users 24 hours a day, seven days a week. Website typically use HTML. To format and present information and to provide navigational facilities that make it easy for the user to move within the site and around the web.
